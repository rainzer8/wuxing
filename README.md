# 润泽-制化方程：五行生克动力学模型

本代码库包含论文《五行生克关系的动力学建模：一种基于微分方程的中医理论形式化探索》的数值模拟部分。通过求解五维线性微分方程组，重现健康状态的自愈过程、病理状态的持续振荡以及自稳力变化导致的分岔现象。

## 文件说明
- `run_all.py`：主脚本，运行后生成论文中的图1、图2、图3，保存于 `images/` 文件夹。
- `requirements.txt`：所需 Python 包列表。

## 运行方法
1. 安装依赖：`pip install -r requirements.txt`
2. 运行脚本：`python run_all.py`

## 结果
- `figure1_healthy.png`：健康状态下系统受扰后回归平衡。
- `figure2_pathological.png`：病理状态下系统无法回归，呈现持续振荡。
- `figure3_bifurcation.png`：自稳力 γ 变化时的分岔图，显示临界阈值。
